# GradProject

Bu repo bir Unity projesidir. Unity’nin standart klasör yapısı (Assets, Packages, ProjectSettings) bulunur.

## Nasıl açılır?
1. Unity Hub’ı açın
2. `Add` / `Open` ile bu repo klasörünü seçin
3. Projeyi Unity içinde açın

## Çalıştırma
- Unity içinde sahneyi (Scene) açtıktan sonra `Play` ile çalıştırabilirsiniz.
- İsterseniz Build almak için:
  - `File > Build Settings` üzerinden platform seçip `Build` alabilirsiniz.

## Not
Repo içinde `StartUnity_EnglishCulture.bat` dosyası var. Windows’ta Unity’yi belirli ayarlarla başlatmak için kullanılabilir.
